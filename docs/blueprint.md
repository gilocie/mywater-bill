# **App Name**: MyWater Bill

## Core Features:

- Tri-Role Authentication Suite: Secure portal for Super Admins, district-specific Staff, and Customers using Meter Number/PIN verification.
- District-Partitioned Management: Permission system ensuring Staff can only access and manage customers within their assigned Malawi district.
- Utility Billing Engine: Tool to calculate bills by multiplying liters consumed by the global water rate, featuring automated invoice and receipt generation.
- Customer Wallet & Auto-Pay: Internal wallet for balance deposits with simulated mobile money integration and automated deduction for pending bills.
- Bulk Data & Reporting Tool: CSV import functionality for bulk customer creation and export capability for regional revenue and payment reports.
- Meter-Based Customer Dashboard: Interactive view for customers to track consumption history, status, and download printable transaction receipts.
- Operational Notification Simulation: Visual and console-based alerts simulating SMS/WhatsApp notifications for invoices, disconnections, and low wallet balances.

## Style Guidelines:

- Primary color: Sophisticated Royal Blue (#2e59cf) symbolizing trust and utility management.
- Background color: Cool Gray-White (#f7f8fb) to ensure a clean, modern administrative workspace.
- Accent color: Bright Cerulean (#1aa7d8) for active buttons and interactive dashboard elements.
- Font: 'Inter' throughout the application for its neutral, machined, and objective look, perfect for data-heavy utility tables.
- Precision line icons representing utility metrics (water drops, wallets, maps) for professional visual cues.
- Sidebar-driven responsive dashboard featuring data grids with robust filtering for regions like Blantyre and Lilongwe.
- Micro-transitions on data refresh and subtle 'Auto Deduction' badges that glow when sufficient wallet funds exist.
